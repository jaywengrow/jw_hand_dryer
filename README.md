# JwHandDryer

Dry your hands from the terminal!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jw_hand_dryer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jw_hand_dryer
